Matches anywhere in a tab's URL. Use * as a wildcard for any text, e.g. github.com/*/pull/*. No * = plain substring match.
Jumps to a matching open Safari tab, or opens the target if none is found. Private targets always open a fresh private window — Safari hides private tabs from automation, so existing private tabs cannot be matched.

Requires: Safari with Automation access for Stream Deck; the private window option also needs Accessibility access (it sends ⌘⇧N).

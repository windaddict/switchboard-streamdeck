Rotate the dial to cycle the windows of the tmux session in the frontmost macOS window; press to jump back to the last window. Tap the touchscreen to widen the scope to ALL sessions — rotation then crosses session boundaries and press jumps to the last session. When iTerm2 isn't frontmost the dial does nothing (the strip shows a dash). The encoder touchscreen shows the current session and window live. There’s nothing to configure.
Requires: a running tmux server (Switchboard talks to tmux directly). Attach tmux in your terminal to see the window change on screen.

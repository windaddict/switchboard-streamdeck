Rotate the dial to move between tmux panes — or, after a press or touchscreen tap toggles the mode, tmux windows. The mode is saved with the button and survives restarts. The dial only drives the tmux session in the frontmost macOS window: when iTerm2 isn't frontmost it does nothing (the strip shows a dash), so it can never move a background terminal.
Requires: a running tmux server (Switchboard talks to tmux directly). For the focused pane to change on screen, tmux should be attached in your terminal.

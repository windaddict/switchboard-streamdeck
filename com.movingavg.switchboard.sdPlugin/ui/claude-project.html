The key face is live: the spark shows whether Claude Code is working (amber, turning) or waiting for your input (white, still) in this project — wherever it runs (tmux, iTerm2, or Terminal). The bar lights up when your keystrokes would land in that session. Pressing raises the hosting window.

Tip: hold the key for ½ second while the session's terminal is frontmost to capture its project into this button (✓ flashes when saved).

Requires: Automation access for iTerm2 and/or Terminal (macOS prompts on first use). Detection itself needs no permissions.
